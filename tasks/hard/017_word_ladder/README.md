## 任务

请实现 `solution.py` 中的 `ladder_length` 函数。

给定两个单词 `begin_word` 和 `end_word`，以及一个字典 `word_list`，找到从 `begin_word` 到 `end_word` 的最短转换序列的长度。转换规则：每次只能改变一个字母；每个转换后的单词必须在字典中。

## 要求

- 返回最短转换序列的长度（包括起始和结束单词）
- 如果不存在转换序列，返回 0
- `begin_word` 不需要在字典中

## 约束

- 不要修改 `test_solution.py`
- 只修改 `solution.py`
